# Random VNF Assignment

Este repositorio contiene un script en Python que simula asignaciones aleatorias de Funciones de Red Virtual (VNF) a nodos de red, considerando restricciones de VNF principal y respaldo.

## Descripción del Código

El código simula múltiples asignaciones aleatorias de VNFs a una lista predefinida de nodos de red. Para cada iteración, se seleccionan VNFs principales y de respaldo aleatorios para cada nodo, verificando si cumplen con la restricción de que el último carácter de los nombres de los VNFs principales y de respaldo no sea el mismo.

Al final de la ejecución, se muestra el número total de asignaciones válidas que cumplen con la restricción de los VNFs.

## Uso

El script se ejecuta directamente en Python y no requiere argumentos adicionales. Basta con ejecutar el código para simular las asignaciones.

## Requisitos

El script está escrito en Python y no tiene dependencias externas. Se recomienda utilizar Python 3.x para la ejecución.
